Relação de Receita dos Municípios do Estado de São Paulo

Este sistema utiliza duas Apis. Utiliza a Api do IBGE para a consulta de Estados e Municípios. Porém como a busca ainda é limitada ao Estado de São Paulo, este campo é restrito ao mesmo, no entando todos os municípios do mesmo são originados da API do IBGE.

A segunda API é a api do do Portal da Transparência do Estado de São Paulo, baseado na escolha do usuário do Site. é possível consultar a receita de um município em determinado mês. Os dados disponíveis são : órgão referente, fonte de receita, origem de recurso e o valor.
